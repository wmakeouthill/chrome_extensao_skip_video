Política de Privacidade

Auto Skip Video - Pular Vídeos Automaticamente

Última atualização: Janeiro 2025

1. Introdução

Esta Política de Privacidade descreve como a extensão "Auto Skip Video - Pular Vídeos Automaticamente" ("nós", "nosso" ou "extensão") coleta, usa e protege suas informações.

2. Informações Coletadas

2.1. Dados NÃO Coletados

A extensão Auto Skip Video NÃO coleta, armazena ou transmite nenhum dos seguintes dados:

  • ❌ Informações pessoais identificáveis
  • ❌ Dados de navegação ou histórico
  • ❌ Informações de vídeos assistidos
  • ❌ Credenciais de login
  • ❌ Dados de localização
  • ❌ Endereços IP
  • ❌ Cookies ou dados de sessão
  • ❌ Quaisquer dados para servidores externos

2.2. Dados Armazenados Localmente

A extensão utiliza apenas o armazenamento local do navegador (chrome.storage.sync) para armazenar:

  • ✅ Estado de ativação/desativação da extensão (apenas "ativado" ou "desativado")
  • ✅ Preferências do usuário relacionadas ao funcionamento da extensão

Estes dados são:

  • Armazenados apenas no seu navegador
  • Nunca transmitidos para servidores externos
  • Acessíveis apenas por você
  • Podem ser excluídos a qualquer momento desinstalando a extensão

3. Como Usamos as Informações

A extensão utiliza apenas os dados armazenados localmente para:

  • Manter suas preferências de ativação/desativação entre sessões
  • Funcionar conforme sua configuração pessoal

Não há processamento externo, análise de dados, ou compartilhamento com terceiros.

4. Permissões da Extensão

A extensão solicita as seguintes permissões:

4.1. storage

  • Finalidade: Armazenar localmente sua preferência de ativação/desativação
  • Dados afetados: Apenas estado on/off da extensão
  • Armazenamento: Local no seu navegador

4.2. activeTab

  • Finalidade: Permitir que a extensão interaja com a aba ativa quando você clica no ícone
  • Uso: Apenas quando você ativa/desativa a extensão via popup
  • Dados afetados: Nenhum

4.3. host_permissions (URLs das plataformas)

  • Finalidade: Permitir que o script de conteúdo seja executado nas plataformas suportadas (YouTube, TikTok, Instagram, etc.)
  • Uso: Apenas para injetar o script que detecta quando vídeos terminam e avança automaticamente
  • Dados afetados: Nenhum - o script funciona apenas localmente na página

5. Scripts de Conteúdo

A extensão injeta um script de conteúdo (content.js) nas páginas das plataformas suportadas. Este script:

  • ✅ Executa apenas no seu navegador
  • ✅ Não transmite dados para servidores externos
  • ✅ Não acessa informações pessoais
  • ✅ Apenas detecta quando vídeos terminam e executa ações locais (scroll, teclas simuladas)
  • ✅ Funciona de forma totalmente offline

6. Serviços de Terceiros

A extensão NÃO utiliza nenhum dos seguintes serviços:

  • ❌ Serviços de analytics
  • ❌ Servidores de rastreamento
  • ❌ APIs externas
  • ❌ Serviços de publicidade
  • ❌ Ferramentas de telemetria
  • ❌ Qualquer serviço que colete dados

7. Compartilhamento de Dados

Não compartilhamos, vendemos ou transmitimos dados para terceiros, pois não coletamos dados para compartilhar.

8. Segurança

Embora não coletemos dados pessoais, implementamos as seguintes práticas de segurança:

  • Código aberto e auditável
  • Uso apenas de APIs oficiais do Chrome
  • Armazenamento local seguro via chrome.storage.sync
  • Sem comunicação com servidores externos

9. Seus Direitos

Você tem total controle sobre seus dados:

  • ✅ Excluir dados: Desinstale a extensão para remover todos os dados armazenados
  • ✅ Ajustar configurações: Altere o estado on/off a qualquer momento via popup
  • ✅ Ver dados armazenados: Use chrome://extensions/ → Detalhes → Opções → Visualizar dados armazenados

10. Crianças

A extensão não coleta intencionalmente informações de crianças. Como não coletamos dados pessoais, não há risco para menores.

11. Alterações nesta Política

Se houver alterações significativas nesta política de privacidade, atualizaremos a data de "Última atualização" e notificaremos os usuários através da página da extensão na Chrome Web Store.

12. Código Aberto

O código-fonte desta extensão está disponível publicamente, permitindo que qualquer pessoa verifique que realmente não coletamos ou transmitimos dados.

13. Contato

Se você tiver dúvidas sobre esta política de privacidade, entre em contato através da página da extensão na Chrome Web Store.

Resumo

A extensão Auto Skip Video não coleta dados pessoais, não transmite informações para servidores externos e funciona totalmente localmente no seu navegador. Armazenamos apenas sua preferência de ativação/desativação localmente, que pode ser excluída a qualquer momento.
