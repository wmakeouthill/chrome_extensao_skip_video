Sobre a Extensão

Entenda como funciona o Auto Skip Video e como ele pode melhorar sua experiência de navegação em vídeos curtos.

O que é o Auto Skip Video?

O Auto Skip Video é uma extensão do Chrome que automatiza a navegação entre vídeos curtos, pulando automaticamente para o próximo quando o vídeo atual termina. Ideal para quem consome muito conteúdo em formato de Shorts, Reels e vídeos curtos, proporcionando uma experiência de navegação contínua e fluida.

A extensão foi desenvolvida pensando em usuários que querem navegar por múltiplos vídeos sem precisar interagir manualmente a cada término, economizando tempo e tornando a experiência mais agradável.

Como Funciona?

A extensão utiliza Manifest V3 e é composta por três componentes principais:

1. Background Service Worker

Inicializa o estado padrão da extensão (ativado por padrão) e gerencia as configurações globais.

2. Content Script

Script principal que roda nas páginas das plataformas suportadas. Detecta quando vídeos terminam e executa ações para avançar para o próximo vídeo.

3. Popup Interface

Interface do usuário para ativar/desativar a extensão. Sincroniza o estado com o content script de forma simples e intuitiva.

Fluxo de Funcionamento

  1. Extensão instalada → Background.js inicializa estado (ativado por padrão)
  2. Usuário navega → Content.js é injetado na plataforma suportada
  3. Content.js detecta plataforma → Seleciona handler apropriado
  4. Content.js encontra elemento <video> → Configura listeners
  5. Vídeo termina → Handler específico executa ação (ArrowDown, scroll, etc.)
  6. Próximo vídeo carrega → Processo se repete automaticamente

Detecção de Fim de Vídeo

A extensão utiliza múltiplas estratégias para detectar quando um vídeo termina:

  • Evento 'ended' - Evento nativo do elemento <video>
  • Monitoramento 'timeupdate' - Verifica quando vídeo está próximo do fim (últimos 0.3s ou 99%)
  • Detecção de loop - Identifica quando vídeo volta ao início após terminar

Prevenção de Problemas

Para garantir uma experiência suave, a extensão implementa várias proteções:

  • Flag `isAvancando` - Previne múltiplas execuções simultâneas
  • Rastreamento de ID do vídeo - Evita processar o mesmo vídeo múltiplas vezes
  • Desativação de loop - Garante que vídeos não fiquem em loop antes de avançar
  • Delay configurável - 300ms de delay para garantir que ações sejam processadas

Como Usar

  1. Instale a extensão da Chrome Web Store
  2. Clique no ícone da extensão na barra de ferramentas
  3. Ative o toggle para ligar a funcionalidade
  4. Navegue para qualquer plataforma suportada
  5. Aproveite! A extensão automaticamente pulará para o próximo vídeo quando o atual terminar
